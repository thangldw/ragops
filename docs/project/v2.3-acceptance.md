# v2.3 release integrity acceptance

## Decision

**GITHUB RELEASED; PYPI HOLD.** Owner authorized staged implementation and
stable publication without GitHub Actions while quota is exhausted. GitHub
v2.3.0 is public; PyPI requires a new project-scoped token in the local session.

Additive CLI discoverability and release verification only. Public schemas,
metrics, gate semantics, API behavior, and open-core boundaries are unchanged.
