# Delivery milestones

| Milestone | Outcome | Owner acceptance evidence |
| --- | --- | --- |
| M0 Stable core | Reproducible local release decision | Tests, schemas, CLI report |
| M1 Benchmark credibility | Evaluation depth visible on a realistic Japanese suite | Taxonomy, 30+ cases, benchmark report |
| M2 Reference deployment | A real RAG/agent emits traces evaluated by RAGOps | Demo, source, failure investigation |
| M3 FDE showcase | Product and delivery story can be presented in 10 minutes | Slides, walkthrough, portfolio page |
| M4 Design partner | External team uses RAGOps on its own system | Interview notes and observed workflow |
| M5 Hosted alpha | Multi-user collaboration validates commercial wedge | Security review and pilot acceptance |

Each milestone requires a go/no-go review by the product owner. Completion of
engineering tasks alone does not constitute product acceptance.
