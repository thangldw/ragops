# Support

RAGOps is an open-source project without a support SLA.

- Use GitHub Discussions for questions and ideas.
- Use a GitHub issue for a focused, reproducible defect.
- Include RAGOps/Python/OS versions, the command, minimal synthetic fixture,
  expected behavior, and actual behavior.
- Do not post customer data, credentials, private traces, or proprietary prompts.
- Feedback does not imply a delivery commitment.

Use the stable `1.1.0` package for published behavior and consult the
[documentation map](docs/README.md) for current guidance.

Security reports must follow [SECURITY.md](SECURITY.md), not a public issue.
