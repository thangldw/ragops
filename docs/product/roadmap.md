# Roadmap

## Next outcomes

- **Activate:** a new user reaches a reviewable blocked regression in under
  three minutes with `uvx ragops demo`.
- **Adopt:** a repository adds the gate in 5–10 YAML lines and receives useful
  metric, block-reason, and HTML evidence on its pull request.
- **Trust:** the Failure Zoo grows from reviewed incidents while every specimen
  keeps a passing control and an explicit claim boundary.

## Discovery priorities

- Identify which CI/review surface creates the most adoption friction.
- Calibrate evaluator policies against qualified human labels.
- Calibrate fixed and sequential statistical policies on reviewed replay
  bundles; measure false-pass, false-block, cost, and stopping behavior.
- Review supported Python versions, provider adapters, and release automation
  for drift before the next package version.

## Deferred hosted hypothesis

Hosted collaboration is not committed roadmap work. It requires evidence of
demand plus separate decisions for identity, tenant isolation, retention,
security, legal, operations, billing, and support. The local control-plane alpha
must not be marketed as production SaaS.
