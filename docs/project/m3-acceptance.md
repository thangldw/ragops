# M3 acceptance — FDE showcase

## Decision

**Accepted as a public portfolio-ready case study.** GitHub Pages is enabled;
the deployment workflow publishes the standalone project site from `site/`.

## Evidence

- Responsive static case-study site with semantic navigation and skip link.
- Discovery, architecture, evaluation, safety, rollout, and decision narrative.
- All displayed metrics reproduce from committed experiment traces.
- Clear synthetic-data and non-production limitations.
- GitHub Pages deployment workflow.
- Browser-verified desktop and 390px mobile layouts with no horizontal overflow.
- Ten-minute presentation outline, five-minute demo walkthrough/video script,
  and personal-portfolio integration handoff.

## Acceptance guardrails

- No unmeasured ROI or adoption claim.
- No claim that lexical claim support is semantic correctness.
- No claim that simulated role lists are production ACL.
- Repository and experiment links remain public and reviewable.
