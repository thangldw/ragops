# v2.4 adoption experience acceptance

## Decision

**READY FOR OWNER ACCEPTANCE; PYPI PROMOTION ON HOLD.** The selected Product
Design direction is publicly available as the developer-proof path from install
to release decision. GitHub Release and GitHub Pages were published without
GitHub Actions; PyPI v2.4.0 remains unpublished until the owner supplies a
project-scoped token locally.

## Contract impact

Presentation-only and backwards compatible. No public evaluation contract,
schema, metric, gate, CLI/API, or open-core boundary changes.

## Limitations

- Visual polish does not prove user acquisition or activation improvement.
- Historical technical records are intentionally not rewritten.
- PyPI byte-for-byte parity cannot be recorded until owner-authorized token
  promotion completes.
- Security testing remains deferred by owner instruction.
