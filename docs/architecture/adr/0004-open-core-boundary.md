# ADR 0004: Preserve a complete open-source release-decision loop

- Status: Accepted
- Date: 2026-07-12

## Context

RAGOps may later support a commercial hosted product. Moving essential
evaluation semantics behind a service would weaken trust and adoption.

## Decision

Scenario contracts, local execution, deterministic evaluators, regression
gates, portable reports, and plugin interfaces remain open source. Commercial
value may be created through collaboration, governance, operations, managed
connectors, and enterprise support.

## Consequences

Community users retain a complete local workflow. Hosted differentiation must
come from team-scale operational value rather than artificial core limits.
